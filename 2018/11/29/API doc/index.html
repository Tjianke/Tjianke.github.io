查询指定单词词根接口

1. 查询指定单词词根

接口功能

给定单词，返回其所有词根

URL

47.101.167.9/query?word=Blahblah

支持格式

JSON

HTTP请求方式

GET

请求参数

参数 必选 类型 说明
word ture string 查询的单词

返回字段

返回字段 字段类型 说明
status int 返回结果状态。0：精准匹配；1：无准确结果；2 ： 错误
etymology_breakdown list 单词词根
similar_words list 相似单词
meaning string 解释

接口示例

地址：47.101.167.9/query?search=Blahblah

{
“status_code”: 0,
“etymology_breakdown”: [discutere, dis, quatere],
“similar_words”: [“talk”, “communicate”]
“meaning”: “blah blah blah”
}

1. 模糊查询指定单词

接口功能

URL

47.101.167.9/dictionary?word=Blahblah

支持格式

JSON

HTTP请求方式

GET

请求参数

参数 必选 类型 说明
word ture string 查询的单词

返回字段

返回字段 字段类型 说明
status int 返回结果状态。0：精准匹配；1：无准确结果；2 ： 错误
similar_words list 相似单词按照相关度排列
